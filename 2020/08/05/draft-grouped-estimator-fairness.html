We're going to try several samplers: over-sampling, under-sampling and no-sampling together with several algorithms: logistic regression, lightgbm, random forest and xgboost.

By no-sampling we mean not addressing the imbalance.

def all_but_first_column(X):
    return X[:, 1:]

scalers = {
    'standardscaler' : StandardScaler(),
    'grouped' : GroupedTransformer(StandardScaler(), groups=[0])
}

learners = {
    'logistic': LogisticRegression(),
    'lgbm': LGBMClassifier(random_state=42),
    'rf': RandomForestClassifier(random_state=42),
    'xgb': XGBClassifier(random_state=42)
    
}

Performance comparison 

coefficients = list()
intercepts = list()

for learner_name, learner in learners.items():
    

    for scaler_name, scaler in scalers.items():

        # Define learner
        if scaler_name == 'grouped':
            # No need for scaling deleting the group column
            pipe = Pipeline([
                ('oh', OrdinalEncoder()),
                ('scaler', scaler),
                ('learner', learner)
            ])
            
        else:
            # Delete group column
            pipe = Pipeline([
                ('oh', OrdinalEncoder()),
                ('scaler', scaler),
                ('dropper', FunctionTransformer(all_but_first_column)),
                ('learner', learner)
            ])

        
        pipe.fit(df_train_x, df_train_y)
        
        # Debug
        # print(pipe[:-1].fit_transform(df_train_x).shape)
        
        # Compute ROC AUC
        test_roc = roc_auc_score(df_test_y, pipe.predict_proba(df_test_x)[:,1]).round(4)
        eq_score = equal_opportunity_score(
            sensitive_column='black'
        )(pipe, df_test_x, df_test_y).round(4)
        
        print(
            f'ROC test {scaler_name}, {learner_name}: {test_roc}'
        )
        
        print(
            f'Equal score test {scaler_name}, {learner_name}: {eq_score}'
        )
                
        # For logistic regression, save coefficients and intercept
        if learner_name == 'logistic':

            intercept = pipe[-1].intercept_[0].round(3)

            coefficients.append(
                pipe[-1].coef_[0]
            )

            intercepts.append(intercept)
    print("----------")

Logistic regression case

In the logistic regression case, we can have a look at the coefficients to try to understand what over and under-sampling are doing.

# Prepare coefficients dataframe

coefficients_df = pd.DataFrame(coefficients).T
intercepts_df = pd.DataFrame(np.array(intercepts)).T

coefficients_df.columns = scalers.keys()

coefficients_df = intercepts_df.append(coefficients_df)

# Clean data to plot

coefficents_long = coefficients_df.melt(var_name='sampling', value_name='coefficient')

coefficents_long['index'] = list(range(coefficients_df.shape[0])) * 2

We can see that most coefficients are very similar in the 3 cases, but the first coefficient changes significantly (in this case, the intercept). The effect on the coefficients of under or over-sampling is increasing the intercept. This completely makes sense - the intercept represents, in a way, the fraction of cases with target 1. By changing this fraction we are artificiallt increasing the intercept.
